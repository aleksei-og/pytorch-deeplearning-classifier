# pytorch-deeplearning-classifier
Проект по созданию многозадачной нейронной сети для предсказания качества вина. Реализует сложную архитектуру с тремя параллельными ветвями (Bottleneck, Inverted Bottleneck, Regular), использует взвешенную функцию потерь для борьбы с дисбалансом классов и достигает целевой метрики F1-score ≥ 40% через оптимизацию гиперпараметров.
# Multi-Branch MLP for Wine Quality Classification

##  Цель проекта
Разработать и обучить модель, которая преодолеет проблему дисбаланса классов в наборе данных о качестве вина и достигнет стабильно высоких результатов по метрике F1.

**Основные задачи:**
1.  Реализовать три типа MLP-блоков: **Bottleneck**, **Inverted Bottleneck** и **Regular**.
2.  Построить **Multi-Branch архитектуру**, объединяющую эти блоки параллельно.
3.  Применить **weighted loss function** для компенсации дисбаланса классов.
4.  Провести подбор гиперпараметров для оптимизации производительности модели.
##  Запуск
Для запуска в среду colab нужно загрузить все .py файлы данного проекта.
Если хотите быстрое выполнение проекта, то шаг с подбором гиперпараметров нужно пропустить.
